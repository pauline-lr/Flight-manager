AJOUT D'UN VOL 
	Numéro de vol
		combi de 2 lettres et 4 chiffres (pas d'espaces, de point, de tiret, etc) 
	
			Regex pour le numéro d'un vol 
			\b[A-z][A-z]\d{4}$

		Unique dans la BD

	Date de départ 
		- à partir d'aujourd'hui

	Heure de départ 
		- si date départ == aujourd'hui => pas d'heure antérieure à celle actuelle 

	Date d'arrivée 
		- doit être ultérieure ou le même jour que la date de départ 

	Heure d'arrivée 
		- Si date d'arrivée == date de départ => doit être ultérieure à l'heure de départ 

NB : Pour les dates et les heures si l'utilisateur entre autre chose ou dans un mauvais format, le programme l'efface automatiquement pour remettre l'heure et la date du jour

	Aéroport de départ/arrivée
		[Pas d'erreur d'encodage possible : JComboBox]

	Terminal 
		D'office une lettre Majuscule
			Regex : \b[A-Z]$ 		

	Porte 
		grâce au spinnerNumberModel d'office un chiffre

	Avion
		[Pas d'erreur d'encodage possible : JComboBox]

	Pilote 
		[Pas d'erreur d'encodage possible : JComboBox]
